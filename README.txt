
Initial Setup:
1. Database Creation:
   Before starting the application, ensure you have a MySQL server running. Create a database for the student management system.

2. SQL File Execution:
   After creating the database, use the provided SQL file to set up the necessary tables and relationships. This file will also contain statements to populate the database with some initial data.

3. Install Requirements:
   Ensure you have Python installed on your system. Then, install the necessary Python libraries by running:
   ```
   pip install flask flask_sqlalchemy pymysql
   ```

**Running the Application:

1. Navigate to the Directory:
   Use the terminal or command prompt and navigate to the directory containing the `main.py` file.

2. Start the Application:
   Once in the directory, run the command:
   ```
   python main.py
   ```

3. After executing the command, the Flask web application should start, and you can access it by opening a browser and going to `http://localhost:5000/`.
